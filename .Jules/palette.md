## 2025-05-15 - [Accessibility and Copy to Clipboard Enhancement]
**Learning:** Adding ARIA attributes like `aria-live="polite"` and `role="alert"` provides immediate feedback to screen reader users without interrupting their flow. Semantic HTML (`header`, `main`, `section`) is essential for structural clarity and navigation via assistive technology.
**Action:** Always wrap key page sections in semantic tags and use ARIA live regions for async status updates. Ensure icon-only or text-only action buttons have descriptive `aria-label` attributes.
